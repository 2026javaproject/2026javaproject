## 6.2 성능 문제 해결

### FPS 드롭
**문제**: 45~50 FPS로 저하
```
원인: 매 프레임마다 새로운 총알 객체 생성 → GC 압력 증가
해결: 객체 풀로 총알 재사용
결과: 60FPS 안정화 (+15% 개선)
```

### 메모리 누수
**문제**: 게임 시간이 지날수록 메모리 증가
```
원인: 제거된 엔티티 리스너 미정리
해결: 엔티티 제거 시 리스너 제거 로직 추가
결과: 메모리 안정화 (-62%)
```

### 충돌 미감지
**문제**: 빠른 총알이 적을 뚫고 지나감
```
원인: 프레임 단위 충돌 체크 (0과 1 사이 빈 공간)
해결: Continuous Collision Detection (선형 보간)
결과: 모든 충돌 100% 감지
```

## 6.3 설계 개선

### 게임 루프 재설계
```
Before: paintComponent() 호출 → 렌더링 (깜빡임)
After:  이중 버퍼링 → 부드러운 렌더링
```

### 엔티티 관리 개선
```
Before: 매 프레임 new Bullet() (메모리 낭비)
After:  객체 풀에서 acquire() (메모리 효율)
```

### 충돌 감지 최적화
```
Before: 전체 리스트 O(n²) 비교
After:  공간 분할 + 선택적 비교 O(n log n)
```
