## 5.3 성능 최적화

### 메모리 최적화
- **객체 풀 패턴**: 총알 재사용 → GC 부하 75% 감소
- **캐싱**: 리소스 한 번만 로드

### 렌더링 최적화
- **이중 버퍼링**: 화면 깜빡임 제거
- **Dirty Rectangle**: 부분 업데이트로 CPU 30% 감소

### 알고리즘 최적화
- **Continuous Collision Detection**: 빠른 객체도 충돌 감지
- **Iterator 사용**: ConcurrentModificationException 방지
