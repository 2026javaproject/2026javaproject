## 9.1 사용 목적

- 코드 생성

## 9.2 초기 설계 단계

### 프롬프트 1: 프로젝트 구조 설계
```
나는 자바 스윙 기반으로 2D 횡스크롤 방식 전투기 게임을 개발할꺼야
언어는 자바, 인텔리제이랑 깃허브로 개발하고
패키지는 유틸, 메인, 화면, 엔티티, 매니저, ui 이렇게 나눌건데
각 패키지가 뭘 담당하는지 정리해줘
```

### 프롬프트 2: 요구사항 정리
```
게임 기능 17개 정도 되는데 F-01~F-17로 번호 매겨서
표로 정리해줘. 누가 담당할지(강민석/임대훈)도 같이 넣어줘
```

---

## 9.3 구현 단계

### 프롬프트 3: 게임 루프 구현
```
스윙에서 60fps로 게임 루프 돌리고 싶은데
update랑 paintComponent 나눠서 짜줘
화면 깜빡이는거 없게 더블버퍼링도 적용해줘
```

### 프롬프트 4: Player 클래스
```
플레이어 클래스 만들어줘
방향키나 wasd로 이동되게 하고 속도는 5
스페이스 안눌러도 자동으로 12프레임마다 발사되게
맞으면 90프레임동안 무적되게 하고
화면 밖으로 안나가게 막아줘
```

### 프롬프트 5: 충돌 감지
```
총알이랑 적 충돌하는거 어떻게 처리해야돼?
Rectangle.intersects() 써서
총알-적, 적총알-플레이어, 아이템-플레이어
이렇게 다 체크하게 해줘
```

### 프롬프트 6: 적 클래스 구조
```
적 종류를 여러개 만들고 싶은데
Enemy 추상클래스 하나 두고
직선으로 내려오는거, 사인파로 움직이는거, 플레이어 쫓아오는거
이렇게 3종류 + 보스(2페이즈)까지 만들어줘
```

### 프롬프트 7: 점수/스테이지 관리
```
점수랑 스테이지 관리하는거 ScoreManager로 싱글톤 만들어줘
적마다 점수 다르게, 일정 마리 잡으면 스테이지 넘어가고
최고점수는 파일로 저장/불러오기 되게 해줘
```

### 프롬프트 8: 메인 메뉴 패널 구현
```
MainMenuPanel에 시작, 최고점수, 종료 버튼 만들어줘.
마우스 클릭과 키보드(↑↓, Enter, Space)로 조작되고,
버튼에 마우스 올리면 색 바뀌고,
최고점수 파일 없으면 BEST: 0 표시해줘.
버튼은 사각형 테두리로 위에서 아래로 표시.
```

### 프롬프트 9: 게임 패널 구현
```
GamePanel을 JPanel과 Runnable로 만들어줘.
게임 루프 실행하고, ESC 누르면 일시정지/재개,
적 파괴되면 폭발 효과,
보스 등장 전에 HUD랑 연동해서 WARNING 빨간 글씨 반복 출력.
```

### 프롬프트 10: 게임 중지 효과 확인
```
GamePanel과 HUD가 아래 조건 만족하는지 확인해줘.
GamePanel: ESC 누르면 paused, 모든 이동 중단, PAUSED 표시, ESC 재입력시 복귀, 
보스 등장 조건 되면 BOSS_WARNING 전환
HUD: WARNING 출력시 빨간색 깜빡임, 150프레임 동안 표시, 경고 중엔 적 이동/충돌 중단
```

### 프롬프트 11: 게임오버 패널 확인
```
GameOverPanel이 아래 기능들을 구현하고 있는지 확인해줘.
최종 점수 표시
최고 점수 표시
신기록시 NEW RECORD 황금색 표시
RESTART 버튼
MENU 버튼
```

### 프롬프트 12: 최고점수 파일 저장
```
HighScoreFileIO.java 만들어줘.
게임오버시 현재 점수랑 최고점수 비교
신기록시 highscore.dat에 저장
메인메뉴 진입시 파일에서 불러오기
파일 없으면 0 반환
```

### 프롬프트 12: 보스 출현 조건 수정
```
보스 출현 조건 수정해줘.
1스테이지: 일반적 11마리 처치시 출현
보스 처치시 스테이지 1씩 증가
2스테이지 12마리, 3스테이지 13마리...
29스테이지 39마리, 이후 39마리 유지
```

## 9.4 최적화 단계

### 프롬프트 13: 객체 풀
```
총알 생성/삭제 너무 자주 일어나서 그런지 좀 버벅이는거같은데
객체 풀 패턴으로 재사용하게 만들어줘
acquire, release 이런식으로
```

### 프롬프트 14: 성능 최적화
```
fps가 60 밑으로 자주 떨어지는데 안정화 좀 시켜줘
더블버퍼링 다시 점검하고
GC 부담 줄이고 쓸데없이 객체 새로 만드는거 없애줘
```

### 프롬프트 15: 빠른 총알 충돌 놓치는 문제
```
총알 속도가 빠르니까 가끔 적 그냥 통과하는데
이거 CCD(Continuous Collision Detection)로 고쳐줘
중간 경로도 체크하게
```
