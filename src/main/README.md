# 2026javaproject 변경 및 작성 내역

## 주요 변경사항 및 기능 추가

1. **패키지 경로 수정**
   - GameConstants, HighScoreFileIO 등 import 경로를 `util` → `com.pilot.util`로 일괄 수정

2. **보스 출현 및 스테이지 증가 로직 개선**
   - 1스테이지: 일반 적 11마리 처치 시 보스 출현
   - 2~29스테이지: 스테이지마다 1마리씩 증가(최대 39마리)
   - 30스테이지 이상: 39마리 처치 시 보스 출현, 보스 처치 시 스테이지 1 증가

3. **HUD(게임 UI) 개선**
   - 하트(HP) 테두리 제거, 하트 모양 개선
   - 하얀색 원(불필요한 테두리) 제거
   - 하트 크기 조정 가능 안내

4. **적 처치 수(KILL) 표시 기능 추가**
   - HUD에 "KILL: n" 형태로 현재 잡은 적 수 표시
   - STAGE 글씨 아래에 위치

5. **HUD 텍스트 색상 통일**
   - STAGE 글씨를 하얀색으로 변경

6. **.class, out/ 등 빌드 산출물은 깃허브에 올리지 않음**
   - src/의 .java 파일과 프로젝트 설정 파일만 관리
