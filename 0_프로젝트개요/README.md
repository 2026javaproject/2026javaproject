## 프로젝트 목표

본 프로젝트는 Java Swing을 활용하여 실시간 객체 처리와 그래픽 렌더링 기술을 구현하는
2D 횡스크롤 슈팅 게임 개발을 목표로 합니다.(배울예정)

- 제한된 시간(약7주) 안에 핵심 로직 구현부터 최종 배포(JAR)까지 완료
- 유지보수가 용이한 객체지향(OOP) 설계 적용
- MVC 패턴 기반으로 로직과 화면을 분리하여 협업 효율 극대화
- 게임 루프, 충돌 감지, 파일 I/O 등 핵심 개념을 실습

---

## 게임 기본 규칙

- 플레이어는 전투기를 조종해 적을 격추하며 점수를 획득한다
- 일정 수의 적을 처치하면 보스가 등장한다
- 보스를 처치하면 다음 스테이지로 진행된다
- 목숨(HP)이 0이 되면 게임오버, 최고점수는 파일에 저장된다

---

## 패키지 요약

- util/       → 전역 상수, 파일 I/O
- 메인/       → 프로그램 진입점, 화면 전환
- 화면/       → 게임 화면, 메인메뉴, 게임오버
- 적/         → 플레이어, 적, 총알, 아이템 오브젝트
- manager/    → 적 스폰, 충돌 감지, 점수, 아이템 관리
- UI/         → HUD (점수, HP, 스테이지 표시)
