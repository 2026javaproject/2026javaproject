# ✈ Pilot

> Java Swing 기반 2D 횡스크롤 전투기 슈팅 게임

## 프로젝트 소개
Pilot은 Java와 Swing을 사용하여 구현한 2D 횡스크롤 방식의 전투기 슈팅 게임입니다.

## 주요 기능
| 기능 | 설명 |
|------|------|
| 플레이어 이동 | 방향키(↑↓←→ / WASD)로 상하좌우 이동 |
| 자동 총알 발사 | 일정 주기마다 총알 자동 연속 발사 |
| 적 비행기 3종 | 직선 / 사인파 / 플레이어 추적 이동 패턴 |
| HP(목숨) 시스템 | 목숨 3개, 충돌 시 -1, 0이면 게임오버 |
| 스테이지 시스템 | 처치 수 누적에 따른 스테이지 진행 |
| 보스 전투 | 스테이지 마지막 보스 등장, 체력바 표시 |
| 파워업 아이템 | 속도 증가 / 다중 총알 / 실드 |
| 최고점수 저장 | 파일로 최고점수 영속 저장 및 불러오기 |

## 실행 방법
```
java -jar Pilot.jar
```
Java 17 이상 필요
